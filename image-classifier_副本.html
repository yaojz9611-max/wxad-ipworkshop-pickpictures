图片分类标注工具query 图人工分类

我要标注
上传 Excel，对 query 图做分类、删除整理。此模式下大类（IP）顺序锁定，不可调整。
我要合并他人结果
上传多份大家导出的 JSON，自动按 option_index 合并去重。可继续删图，并在此调整最终大类顺序。
我要检查最终结果
上传 1 份最终 JSON，只读查看图片与分组，确认结果是否符合预期。不可编辑。

使用指引
本工具用于对 query 图做人工分类与标注，全程在你本地浏览器运行，不上传任何数据。先在上方选择一个模式开始。
1
选模式
标注（自己筛）/ 合并（汇总他人结果）/ 检查（查看最终 JSON），按需求点上方对应卡片。
2
上传文件
标注传 Excel；合并传多份 JSON；检查传 1 份 JSON。系统按 IP 自动分大类（顶部 Tab），每类下分 Male / Female。
3
整理标注
不合格的图点「删除」剔除（可恢复）；点图可放大。大类顺序仅在「合并」模式可调，标注模式锁定。
4
导出 JSON
点右上「导出 JSON」，填写名字和负责的 option_index，文件按此自动命名，方便汇总区分。
多人协作怎么用
组长先分工：每人负责一个或几个大类（option_index）。各自用「我要标注」模式打开同一份 Excel，只筛自己负责的大类，导出时填「修改人 + 负责的 option_index」。最后组长用「我要合并他人结果」模式上传所有人的 JSON，自动汇总——分头干、不打架。
重要分批标注时，请勿调整大类（IP）顺序
大类顺序决定最终排序。所以「我要标注」模式已锁定顺序调整，每人只做删除 / 整理。顺序统一留到最后：由组长在「我要合并他人结果」模式里统一调整后导出最终版。
